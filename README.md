# C++的结构化封装算法模板

此仓库为一个算法竞赛模板库，语言为C++，其中的模板代码为用结构体或类封装的算法模板代码
- 用结构体或类进行封装，其旨在让某些功能更方便调用，让模板更容易调试，代码结构更加清晰
- 每份模板都有对应的一个或多个模板题目并在代码开头的注释说明了模板题目来源与链接
- 所有模板均通过模板题目测试，正确性无误，可放心使用
- **注：此模板库目前只保留作者所学过的模板中适合进行封装的模板,并不包含所有常用算法模板**
